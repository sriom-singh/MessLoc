# MessLoc

## One-Line Description

MessLoc is a location-based platform that helps students and working professionals discover, compare, and subscribe to affordable nearby mess services.

## Tech Stack

* Frontend: Next.js
* Styling: Tailwind CSS
* Backend: Express.js
* Database: MongoDB
* Authentication: JWT
* Deployment: Vercel + Render
